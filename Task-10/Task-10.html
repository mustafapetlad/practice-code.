Flexbox Properties Used:

  • display: flex - Create a flex container.
  • flex-direction: row - Arranges item in a row.
  • justify-content: space-between - Spreads items evenly with space between.
  • align-item: stretch - Default; iteams stretch to container height.
  • flex-wrap: wrap - Allows iteams to wrap to a new line if needed.
  • gap - Controls space between iteams.
